Creating Visualizations

Generate custom plots and images to visualize performance.

Overview

Visualizations are Python functions that return a PIL Image. These images are displayed in the Comparison View UI.

Available Libraries

  • PIL (Pillow) - Image, ImageDraw
  • matplotlib.pyplot (as plt)
  • numpy (as np)
  • io - BytesIO

Examples

Drawing with PIL

Fast and simple drawing (rectangles, text, etc.).

def draw_box(value):
    from PIL import Image, ImageDraw
    
    # Create a new image
    img = Image.new('RGB', (100, 50), color='white')
    d = ImageDraw.Draw(img)
    
    # Draw text
    d.text((10, 10), f"Val: {value}", fill='black')
    
    return img
Plotting with Matplotlib

Powerful plotting for graphs, histograms, etc. Note the boilerplate to convert the plot to a PIL Image.

def plot_histogram(values):
    from PIL import Image
    import matplotlib.pyplot as plt
    from io import BytesIO
    
    # Create figure
    fig, ax = plt.subplots(figsize=(4, 3))
    ax.hist(values, bins=20)
    ax.set_title("Distribution")
    
    # Save to buffer
    buf = BytesIO()
    fig.savefig(buf, format='png', bbox_inches='tight')
    buf.seek(0)
    plt.close(fig) # IMPORTANT: Close the figure to free memory
    
    # Convert to PIL
    img = Image.open(buf).convert("RGB")
    return img

Aggregated Visualizations

Just like metrics, visualizations can be Aggregated. This is perfect for creating:

  • Overall Error Distribution Histograms
  • Precision-Recall Curves
  • Scatter Plots (e.g., Error vs. Distance)

Simply mark the visualization as "Aggregated" when creating it in the Global Library, and your function will receive lists of values instead of single items.
