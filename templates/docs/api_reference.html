Automation

POST /<project_name>/leaderboard/<id>/upload_submission

Upload a new submission ZIP file to a specific leaderboard.

Parameters (Multipart/Form-Data)
Name Type Description
submission_zip File The ZIP file containing your results.
submission_name String (Optional) Display name for the submission. Defaults to filename.
Example (Python)
import requests

url = "http://localhost:6060/dTOF/leaderboard/1/upload_submission"
files = {'submission_zip': open('results.zip', 'rb')}
data = {'submission_name': 'My Algorithm v1.0'}

r = requests.post(url, files=files, data=data)
print(r.status_code)
GET /<project_name>/leaderboard_metrics_status

Poll for the status of metric calculations.

Parameters (Query Param)
  • leaderboard_id (required): ID of the leaderboard.
Response
{
    "123": {  // Submission ID
        "id": 123,
        "name": "My Algorithm",
        "metrics": {
            "L1_Error": 0.45,
            "Accuracy": "Calculating..." 
        }
    }
}
