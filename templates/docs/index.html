Bench-Hub

A flexible, self-hosted platform for benchmarking system performance.

What is Bench-Hub?

Bench-Hub is a tool designed to help researchers and engineers compare different algorithms or systems (Submissions) against a Ground Truth (Dataset). It provides:

  • Leaderboards: Rank submissions based on custom metrics.
  • Deep Comparison: Side-by-side view of results at the sample level.
  • Customizability: Write your own metrics and visualizations in Python.

Core Architecture

graph LR P[Project] --> L[Leaderboard] L --> M[Metrics] L --> V[Visualizations] L --> D[Dataset] L --> S1[Submission 1] L --> S2[Submission 2] S1 --> Comparison S2 --> Comparison

Everything in Bench-Hub starts with a Project. Within a project, you create Leaderboards. Each leaderboard is tied to specific Metrics and Visualizations that define how success is measured.
