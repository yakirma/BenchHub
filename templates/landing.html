How it works

Three steps from idea to ranked leaderboard.

1. Pick a dataset

Use your own data or pick one from the public catalog. Images, scalars, depth maps, audio — BenchHub handles the common ML data types.

2. Define metrics in Python

Bring your own scoring code — any function that takes ground truth + prediction and returns a number. Per-sample or aggregated. No DSL to learn.

3. Submit, get ranked

Send your predictions, BenchHub scores them, and your run lands on the leaderboard next to everyone else's.
