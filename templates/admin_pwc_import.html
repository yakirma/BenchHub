Import from Papers With Code

Admin-only mirror flow. Search the PWC static archive (pwc-archive/evaluation-tables on HuggingFace — the original API is gone), pick a dataset, then choose a benchmark. Mirrored submissions appear alongside verified ones on the LB page but skip the eval pipeline.

The archive is frozen at HuggingFace's PWC acquisition (~early 2024). Useful for bootstrapping canonical leaderboards with literature scores; not a live feed.
